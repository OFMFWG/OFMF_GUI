An OFMF is a management tool that is used to manage fabrics.

The Open Fabrics Managemenet is an implementation for Centralized management of fabrics

This tool:

  • Lists the available fabrics in the large scale system

  • Configures the fabrics by talking with agent

  • Notifies when there are changes to the resource
